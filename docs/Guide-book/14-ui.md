# 第 14 章：UI 层

## 14.1 解决的问题

OpenHarness 需要支持多种交互模式：
1. **交互式终端**：全功能 TUI，支持实时流式渲染
2. **Headless 模式**：脚本/管道场景，纯文本或 JSON 输出
3. **子 Agent Worker**：无头进程间通信
4. **远程前端**：React TUI，支持跨机器访问

## 14.2 三层 UI 架构

```
┌─────────────────────────────────────────┐
│            UI 入口选择器                  │
│  run_repl() → React TUI / Textual TUI   │
│  run_print_mode() → Headless 输出       │
│  run_task_worker() → 子 Agent 通信       │
└─────────────────────────────────────────┘
```

## 14.3 React TUI（推荐）

### 14.3.1 架构

React TUI 使用 **React + Ink** 构建终端 UI，通过 JSON-lines 协议与 Python 后端通信：

```
┌──────────────────────────────────────────────────┐
│               浏览器 / 终端                        │
└──────────────────┬───────────────────────────────┘
                   │
                   ▼
┌──────────────────────────────────────────────────┐
│           React TUI (Ink + React)                 │
│  frontend/terminal/src/                           │
│    index.tsx         ← 入口                        │
│    App.tsx           ← 主应用                      │
│    Transcript.tsx    ← 对话记录                    │
│    Composer.tsx      ← 输入框                      │
│    Sidebar.tsx       ← 侧边栏                      │
└──────────────────┬───────────────────────────────┘
                   │ JSON-lines (stdin/stdout)
                   ▼
┌──────────────────────────────────────────────────┐
│         Python Backend Host                      │
│  ui/backend_host.py                               │
│  ReactBackendHost:                                 │
│    ← 接收 FrontendRequest                         │
│    → 发送 BackendEvent                             │
└──────────────────────────────────────────────────┘
```

### 14.3.2 启动方式

`ui/react_launcher.py` 中的 `launch_react_tui()`：

```python
async def launch_react_tui(backend_command):
    """启动 React TUI。"""
    tsx = _resolve_tsx(frontend_dir)
    process = await asyncio.create_subprocess_exec(
        *tsx, "src/index.tsx",
        cwd=str(frontend_dir),
        env={"OPENHARNESS_FRONTEND_CONFIG": json.dumps({
            "backend_command": backend_command,
            "initial_prompt": None,
            "theme": "default",
        })},
    )
    return await process.wait()
```

### 14.3.3 前后端协议

`ui/protocol.py` 定义了通信协议：

**前端 → 后端（FrontendRequest）**：
```python
FrontendRequest = (
    SubmitLineRequest          # 提交用户输入
    | PermissionResponse       # 权限确认响应
    | QuestionResponse         # 用户提问响应
    | ListSessionsRequest      # 列出会话
    | SelectSessionRequest     # 选择会话
    | SelectCommandRequest     # 选择命令
    | RefreshStateRequest      # 刷新状态
)
```

**后端 → 前端（BackendEvent）**：
```python
BackendEvent = (
    ReadyEvent                 # 后端就绪
    | TranscriptItemEvent      # 对话条目追加
    | AssistantDeltaEvent      # 流式文本增量
    | ToolStartedEvent         # 工具开始
    | ToolCompletedEvent       # 工具完成
    | ErrorEvent               # 错误
    | StatusEvent              # 状态
    | StateSnapshotEvent       # 状态快照
    | TasksSnapshotEvent       # 任务快照
)
```

### 14.3.4 BackendHost

`ui/backend_host.py` 中的 `ReactBackendHost` 实现了协议处理：

```python
class ReactBackendHost:
    async def run(self):
        """主循环：监听 stdin 上的 FrontendRequest。"""
        await self._emit(BackendEvent(type="ready"))
        
        for line in sys.stdin:
            request = json.loads(line)
            
            if request.type == "submit_line":
                await self._handle_submit(request)
            elif request.type == "permission_response":
                await self._handle_permission(request)
            elif request.type == "list_sessions":
                await self._handle_list_sessions(request)
            # ...
    
    async def _handle_submit(self, request):
        """处理用户提交。"""
        async for event in self._bundle.engine.submit_message(request.text):
            if isinstance(event, AssistantTextDelta):
                await self._emit(BackendEvent(
                    type="assistant_delta",
                    text=event.text,
                ))
            elif isinstance(event, ErrorEvent):
                await self._emit(BackendEvent(
                    type="error",
                    message=event.message,
                ))
            # ...
```

```
│  │  /help for commands >______________ ││
│  └─────────────────────────────────────┘│

│  │ Tools 40 │ Memory 128k│ Mode: Plan ││
│  └──────────┴────────────┴────────────┘│
└─────────────────────────────────────────┘
```

事件渲染：

```python
class OpenHarnessTerminalApp(TextualApp):
    def _render_event(self, event):
        if isinstance(event, AssistantTextDelta):
            self._append_text(event.text)
        elif isinstance(event, ToolExecutionStarted):
            payload = json.dumps(event.tool_input)
            self._append_line(f"tool> {event.tool_name} {payload}")
        elif isinstance(event, ToolExecutionCompleted):
            prefix = "tool-error>" if event.is_error else "tool-result>"
            self._append_line(f"{prefix} {event.tool_name}: {event.output}")
        elif isinstance(event, ErrorEvent):
            self._append_line(f"error> {event.message}")
        elif isinstance(event, StatusEvent):
            self._append_line(f"system> {event.message}")
```

## 14.5 Headless 模式

### 14.5.1 Print Mode

`ui/app.py` 中的 `run_print_mode()` 提供三种输出格式：

```python
async def run_print_mode(prompt, output_format="text"):
    bundle = await build_runtime(...)
    
    async def render_event(event):
        if output_format == "text":
            if isinstance(event, AssistantTextDelta):
                sys.stdout.write(event.text)
        elif output_format == "json":
            if isinstance(event, ErrorEvent):
                print(json.dumps({"type": "error", "message": event.message}))
        elif output_format == "stream-json":
            # 每行一个 JSON 事件
            ...
    
    await handle_line(bundle, prompt, render_event=render_event)
```

### 14.5.2 Task Worker

`run_task_worker()` 用于子 Agent 进程：

```python
async def run_task_worker():
    """作为子 Agent 运行，通过 stdin/stdout 协议通信。"""
    bundle = await build_runtime(...)
    
    for line in sys.stdin:
        request = json.loads(line)
        
        if request.type == "task":
            # 接收任务，运行查询，输出结果
            async for event in bundle.engine.submit_message(request.prompt):
                if isinstance(event, AssistantTurnComplete):
                    response = {"type": "result", "output": event.message.text}
                    print(json.dumps(response), flush=True)
                elif isinstance(event, ErrorEvent):
                    response = {"type": "error", "message": event.message}
                    print(json.dumps(response), flush=True)
```

## 14.8 本章小结

UI 层通过**三层架构**提供灵活的交互体验：**React TUI**（基于 Ink 的现代终端 UI）、**Textual TUI**（备选 Python TUI）、**Headless 模式**（脚本/管道/子进程）。所有模式共享相同的 `handle_line()` + 事件流架构，保证行为一致性。
