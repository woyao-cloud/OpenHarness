### 🧠 题型1：三角函数图像与性质

**高考问法**：
> 已知 `f(x) = A sin(ωx + φ) + B` 的部分图像，求解析式，再求单调区间/对称轴/最值。

**解题三步走**：

```
第一步：看图求 A, B, ω, φ
  A = (最大值-最小值)/2
  B = (最大值+最小值)/2
  ω = 2π/T（T由图像相邻零点或最高点间距得到）
  φ = 代入已知点坐标（注意范围）

第二步：求单调区间
  令 -π/2 + 2kπ ≤ ωx+φ ≤ π/2 + 2kπ → 解出x范围（增区间）
  令 π/2 + 2kπ ≤ ωx+φ ≤ 3π/2 + 2kπ → 解出x范围（减区间）

第三步：求最值/对称轴/对称中心
  最大值：ωx+φ = π/2 + 2kπ
  最小值：ωx+φ = 3π/2 + 2kπ
  对称轴：ωx+φ = π/2 + kπ
  对称中心：ωx+φ = kπ
```

**⚠️ 易错点**：
- 平移变换时注意"左加右减"是对 **x** 操作，不是对 **ωx**
- 求φ时注意范围，一般是 `|φ| < π` 或 `0 < φ < π`

---

### 🧠 题型2：三角恒等变换+求值

**核心公式清单（必须烂熟于心）**：

```
【和差公式】
sin(α±β) = sinα cosβ ± cosα sinβ
cos(α±β) = cosα cosβ ∓ sinα sinβ
tan(α±β) = (tanα ± tanβ)/(1 ∓ tanα tanβ)

【倍角公式】
sin2α = 2 sinα cosα
cos2α = cos²α - sin²α = 2cos²α - 1 = 1 - 2sin²α
tan2α = 2tanα/(1-tan²α)

【降幂公式】（非常重要！）
sin²α = (1-cos2α)/2
cos²α = (1+cos2α)/2

【辅助角公式】（必考！）
a sinθ + b cosθ = √(a²+b²) sin(θ+φ)，其中 tanφ = b/a
```

**解题口诀**："**统一角度、统一函数名、统一形式**"
- 不同角 → 用和差/倍角化为同角
- 不同函数名 → 用诱导/辅助角化为同名
- 高次 → 用降幂公式降次

### 🧠 题型3：解三角形（最常考！）

**两个核心工具**：

```
【正弦定理】
a/sinA = b/sinB = c/sinC = 2R（R为外接圆半径）

【余弦定理】
a² = b² + c² - 2bc·cosA
↳ cosA = (b² + c² - a²)/(2bc)

【面积公式】
S = (1/2)bc·sinA = (1/2)ac·sinB = (1/2)ab·sinC
```

**解题套路——看已知条件选定理**：

| 已知条件 | 首选定理 | 为什么 |
|:--|:--:|:--|
| 两角一边 | **正弦定理** | 用正弦求另一边 |
| 两边一夹角 | **余弦定理** | 用余弦求第三边 |
| 三边 | **余弦定理** | 用余弦求角 |
| 两边一对角 | **正弦定理**（注意两解！） | 可能有两解 |
| 边的关系式 | **边化角或角化边** | 统一为角或边 |

**💡 得分套路——万能模板**：

```
题目：在△ABC中，角A,B,C的对边分别为a,b,c，
已知 b·cosC + c·cosB = 2a·cosA

解：
① 边化角（全部换成sin）
   sinB·cosC + sinC·cosB = 2sinA·cosA

② 用和差公式合并左边
   sin(B+C) = 2sinA·cosA

③ 三角形中sin(B+C) = sinA（重要！三角形内角和π）
   sinA = 2sinA·cosA

④ 因为sinA≠0，两边约去sinA
   cosA = 1/2

⑤ 所以 A = π/3
```

**✨ 解三角形大题的高分步骤模板**：

```
【第1问——求角/边】
① 写：由正弦/余弦定理得...
② 代：代入已知数据
③ 算：计算结果
④ 答：所以 A=60° 或 a=3

【第2问——求面积/周长/取值范围】
① 先用第1问结果
② 面积用 S=(1/2)bc·sinA
③ 周长用 a+b+c，用余弦找关系
④ 取值范围用基本不等式或函数法
```

### 🧠 核心公式清单

```
【等差数列】
通项：an = a₁ + (n-1)d
前n项和：Sn = n(a₁+an)/2 = na₁ + n(n-1)d/2
等差中项：a,b,c成等差 → 2b = a+c

【等比数列】
通项：an = a₁·q^(n-1)
前n项和：
  q≠1时：Sn = a₁(1-qⁿ)/(1-q)
  q=1时：Sn = na₁
等比中项：a,b,c成等比 → b² = ac

【常用性质】
等差：若m+n=p+q，则am+an=ap+aq
等比：若m+n=p+q，则am·an=ap·aq
```

---

### 🧠 题型1：等差/等比基本量计算

**💡 核心思路**：**列方程，解方程**

> 题目给两个条件 → 设 a₁ 和 d（或 q）→ 列两个方程 → 解出来 → 一切解决

**示例**：
```
已知{an}是等差数列，a₂=5，a₅=11，求an和S₁₀

解：设首项a₁，公差d
则 a₂ = a₁+d = 5    ①
   a₅ = a₁+4d = 11  ②

②-①：3d=6 → d=2
代入①：a₁=3

所以 an = 3 + (n-1)×2 = 2n+1
S₁₀ = 10×3 + 10×9×2/2 = 30+90 = 120
```

---

### 🧠 题型2：求通项公式（三种常考类型）

**类型①：已知Sn求an**（必考！）

```
万能公式：an = { S₁ (n=1)
                { Sn - S(n-1) (n≥2)

⚠️ 必须分n=1和n≥2讨论！
```

**类型②：递推公式 an+1 = pan + q**（构造等比）

```
形如 an+1 = 2an + 3

解法：构造等比数列
设 an+1 + λ = 2(an + λ)
展开：an+1 = 2an + λ → 对比原式得 λ = 3

所以 {an+3} 是等比数列，首项 a₁+3，公比2
→ an+3 = (a₁+3)·2^(n-1)
→ an = (a₁+3)·2^(n-1) - 3
```

**类型③：an+1 = an + f(n)**（累加法）

```
形如 an+1 = an + 2n+1

解法：
a₂-a₁ = 2×1+1
a₃-a₂ = 2×2+1
...
an-a(n-1) = 2×(n-1)+1

累加得：an - a₁ = 2[1+2+...+(n-1)] + (n-1)
         = 2×n(n-1)/2 + (n-1)
         = n²-1

∴ an = a₁ + n² - 1
```

---

### 🧠 题型3：数列求和（两种必会方法）

**方法①：裂项相消法** ⭐⭐⭐⭐⭐（考得最多！）

```
常见裂项公式（背下来！）：
1/(n(n+1)) = 1/n - 1/(n+1)
1/(n(n+k)) = (1/k)(1/n - 1/(n+k))
1/((2n-1)(2n+1)) = (1/2)(1/(2n-1) - 1/(2n+1))
1/(√(n+1)+√n) = √(n+1) - √n
n/(n+1)! = 1/n! - 1/(n+1)!
```

**💡 裂项口诀**："**分母化乘积，分开减一减，系数别漏掉**"

**方法②：错位相减法** ⭐⭐⭐⭐

```
适用：等差×等比 型数列求和
例如：an = (2n-1)·3ⁿ

步骤：
① 写出Sn = 1·3¹ + 3·3² + 5·3³ + ... + (2n-1)·3ⁿ
② 乘以公比：3Sn = 1·3² + 3·3³ + ... + (2n-3)·3ⁿ + (2n-1)·3ⁿ⁺¹
③ ①-②（错位相减）：
   -2Sn = 3¹ + 2(3²+3³+...+3ⁿ) - (2n-1)·3ⁿ⁺¹
④ 等比求和化简，得出Sn
```

### 🧠 核心定理清单

**平行关系**：
```
线线平行 → 线面平行
  a∥b, a⊄α, b⊂α → a∥α

线面平行 → 线线平行
  a∥α, a⊂β, α∩β=b → a∥b

线线平行 → 面面平行
  a∥β, b∥β, a∩b=P, a⊂α, b⊂α → α∥β
```

**垂直关系**：
```
线线垂直 → 线面垂直
  l⊥a, l⊥b, a∩b=P, a⊂α, b⊂α → l⊥α

线面垂直 → 线线垂直
  l⊥α, a⊂α → l⊥a

线面垂直 → 面面垂直
  l⊥α, l⊂β → α⊥β

面面垂直 → 线面垂直
  α⊥β, α∩β=m, l⊥m, l⊂β → l⊥α
```

---

### 🧠 建系法【三步走】（90%的题都用这个方法）

> **只要题目中有三条两两垂直的线，优先建系！**

**第一步：建系——选原点，定坐标轴**

```
选一个特殊点做原点（通常是顶点或中点）
x,y,z轴沿三条两两垂直的方向建立

❗ 关键：写出所有点的坐标
```

**第二步：求关键向量和法向量**

```
【向量求法】
向量AB = (x₂-x₁, y₂-y₁, z₂-z₁)

【法向量求法——叉乘法】（最快！）
已知平面内两个向量 a=(a1,a2,a3), b=(b1,b2,b3)
法向量 n = a × b

n = (a2·b3 - a3·b2, a3·b1 - a1·b3, a1·b2 - a2·b1)

💡 口诀：x不看，y差z；y不看，z差x；z不看，x差y
   更简单的记忆：n = (a2b3-a3b2, a3b1-a1b3, a1b2-a2b1)
```

**第三步：代入公式求角**

```
【线面角公式】（线与平面所成角）
设直线方向向量为 a，平面法向量为 n
则 sinθ = |a·n| / (|a|·|n|)
  （注意：线面角是sin，不是cos！）

【二面角公式】（两个平面的夹角）
设两个平面的法向量分别为 n₁, n₂
则 cosθ = |n₁·n₂| / (|n₁|·|n₂|)
  （注意：二面角可能是锐角或钝角，看题目要求）

【点到平面的距离】
点P到平面α（法向量n，平面内一点A）的距离：
d = |AP·n| / |n|
```

### 🧠 常见几何体的坐标设定

```
【正方体/长方体】
以顶点为原点，三条棱为坐标轴

【三棱锥】
以底面三角形的一个顶点为原点，
用两条边做x,y轴，高做z轴

【直棱柱】
以下底面中心为原点，
底面两条垂线为x,y轴，侧棱为z轴

【圆柱/圆锥】
以底面中心为原点，底面直径方向为x,y轴，高为z轴
```
